# 이른아침 · 발주관리 PWA — 변경 이력

---

## v2.3.0 · 2026-06-14

### 🆕 신규
- **납품 리스트 사진 AI 자동 매칭** (업로드 탭 상단 추가)
  - 납품 확인서·납품 리스트 사진(JPG/PNG/PDF) 업로드
  - Gemini 2.5 Flash가 이미지 분석 → 등록된 발주목록과 선명·서류번호·품목명 기준 자동 매칭
  - 매칭 결과 카드에서 개별 납품완료 처리 또는 **전체 일괄 납품완료 처리** 가능
  - 납품완료 처리 시 비고에 `[납품사진 자동확인]` 자동 기록
  - 발주목록에 없는 미매칭 항목도 별도 표시

### 🔧 변경
- **대시보드 stat 카드 4개 재편** — 발주 기준 → 납품 실적 기준으로 변경
  | 이전 | 변경 후 |
  |------|---------|
  | 총 발주건수 | 납품 건수 (납품완료 + 부분납품) |
  | 총 발주금액 | 납품 총금액 (반품 차감 실납품 기준) |
  | 총 박스수 (전체 발주) | 납품 박스수 (납품완료 발주 기준) |
  | 선명 수 | 선명 수 (유지) |
- 탑바 로고 배지에 버전 표기 추가 (`발주관리 v2.3`)
- 업로드 탭 헤더 문구 변경: `문서 업로드` → `업로드 (발주서 등록·납품 자동 확인)`
